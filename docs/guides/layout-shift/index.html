Minimising Layout Shift

Rendering JavaScript elements into the DOM (particularly when loading data asynchronously) can cause the page content to jump around the page while loading.

This can be jarring for users (particularly on mobile) and is now factored into Google's metrics, so it's important to consider when adding React components to a page.

Where possible, a placeholder should be included in the area in which a component will be rendered with the same dimensions as the rendered content.
